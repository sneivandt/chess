# ACE - A Chess Engine

A UCI chess engine for Linux and Windows.

## Build

    make

## Run

On Linux, start a game in xboard as white playing against the engine as black.

    ./xboard.sh

## CLI Mode

Enter moves in algebraic notation or some action key for special actions.

Key | Action
----|-------
n   | New Game
q   | Quit
u   | Undo
s   | Search
e   | Evaluate
p   | Perft Test
